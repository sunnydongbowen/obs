- 用于查看当前登录主机的用户终端信息,能看到时间，ip地址。 
```bash
[root@os11 ~]# who
root     pts/0        2021-10-29 10:37 (192.168.1.16)
root     pts/2        2021-10-29 11:34 (192.168.1.16)
```

```bash
[root@centos8 ~]# whoami
root
```
